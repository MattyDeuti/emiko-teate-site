# びわ葉温圧 手あて - ウェブサイト

## 概要

このプロジェクトは、長崎県諫早市にある「びわ葉温圧 手あて」のための完全なウェブサイトです。日本の伝統療法である枇杷葉温圧について紹介し、サロンの情報、施術コース、セラピスト紹介などを提供します。

## 技術仕様

- **HTML5**: セマンティックなマークアップ
- **Tailwind CSS**: レスポンシブデザインとユーティリティクラス
- **カスタムCSS**: 追加のスタイリングとアニメーション
- **Vanilla JavaScript**: インタラクティブ機能
- **Google Fonts**: Noto Sans JP フォント

## ファイル構造

```
one page miko/
├── index.html          # メインページ
├── styles.css          # カスタムスタイルシート
├── script.js           # JavaScript機能
├── images/             # 画像フォルダ
│   ├── Emiko.webp      # セラピスト写真
│   ├── biwa-treatment.webp # 施術写真
│   ├── header.webp     # ヘッダー画像
│   ├── materiel.webp   # 施術器具
│   ├── plantes.webp    # 植物画像
│   ├── icon-*.svg      # 各種アイコン
│   └── ...
├── 404.html            # エラーページ
├── robots.txt          # SEO設定
├── sitemap.xml         # サイトマップ
├── netlify.toml        # Netlify設定
├── _headers            # HTTPヘッダー設定
├── _redirects          # リダイレクト設定
└── README.md           # このファイル
```

### セクション詳細

#### 1. ヒーローセクション
- サロンの紹介とメインメッセージ
- 美しいヘッダー画像
- 自然療法への信念を表現

#### 2. びわ葉温圧について
- 施術の効果説明（細胞の活性化、美容効果、リラクゼーション）
- 自然の力への信念
- 視覚的な説明とアニメーション

#### 3. こんな方ご相談ください
5つのカテゴリーで対象者を明確化：
- 慢性的な肩こり・腰痛に悩んでいる方
- 体質改善や自然療法に興味のある方
- 薬に頼らないセルフケアを求めている方
- リラックスや心の落ち着きを感じたい方
- 女性のお悩みや不眠などに悩んでいる方

#### 4. 施術コース
- びわ葉温圧セッション：60分 3,500円
- 詳細な説明とサービス内容
- 出張サービス情報（500円、長崎・諫早・雲仙・大村市）

#### 5. 施術の流れ
3ステップでの施術プロセス：
1. ヒアリング
2. 準備（枇杷の葉の配置）
3. 施術（温熱療法）

- セラピスト「Emiko」の紹介
